# 🧮 Mein erster JS-Taschenrechner

Ein kleiner, übersichtlicher Web-Taschenrechner, der komplett ohne dicke Frameworks auskommt – einfach nur pures HTML, CSS und JavaScript. 

Ich habe diese App gebaut, um mich praktisch auf meine Wunsch-Ausbildung zum **Fachinformatiker für Anwendungsentwicklung** vorzubereiten. Mir war es wichtig, nicht nur graue Theorie zu lernen, sondern direkt etwas zu programmieren, das man auch anfassen und benutzen kann.

🔗 [Klick hier, um die Live-Demo auszuprobieren!](https://paudelutshab-ai.github.io/Javascript-basics-/)

---

## 🔥 Was der Rechner kann
* **Die Klassiker:** Er beherrscht alle Grundrechenarten (Plus, Minus, Mal, Geteilt).
* **Flüssiges Layout:** Dank CSS Grid passt sich das Design automatisch an. Egal ob auf dem Smartphone in der Bahn oder am Desktop-Monitor – der Rechner sieht überall ordentlich aus.
* **Absturzsicher:** Wenn man versucht, Quatsch einzugeben (wie z.B. `5++2`), fängt der Code das ab und zeigt sauber ein "Error" an, anstatt komplett einzufrieren.

---

## 🛠️ Tech-Stack
* **HTML5** – Für das Grundgerüst der Buttons und des Displays.
* **CSS3** – Für das cleane, dunkle Design und die Grid-Anordnung.
* **JavaScript (ES6)** – Für die gesamte Logik im Hintergrund.

---

## 🧠 Was ich bei diesem Projekt gelernt habe
Das war ein super Projekt, um die Grundlagen zu festigen. Besonders viel mitgenommen habe ich bei:
1. **DOM-Manipulation:** Wie JavaScript überhaupt mit dem HTML-Display spricht, um Zahlen in Echtzeit anzuzeigen.
2. **Fehlerbehandlung:** Ich habe gelernt, wie wichtig `try...catch`-Blöcke sind, um Fehlereingaben der Nutzer elegant abzufangen.
3. **CSS Grid:** Raster-Layouts haben mir früher oft Kopfschmerzen bereitet, aber durch die Anordnung der Tasten habe ich das System endlich richtig verstanden.
